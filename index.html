ACTIVIDAD: APP “ADIVINA EL NÚMERO”

Valor: 250,000 | Fecha límite: hoy antes de las 10:30

¿En qué consiste?

La app debe generar o tener un número secreto, y el usuario deberá intentar adivinarlo escribiendo un número en una caja de texto.

La aplicación debe cumplir con lo siguiente:

  • 1. Tener un título visible.
  • 2. Tener una caja de texto para que el usuario escriba un número.
  • 3. Tener un botón para comprobar si el número es correcto.
  • 4. Mostrar un mensaje que indique:
    • “El número es mayor”
    • “El número es menor”
    • “¡ADIVINASTE!”
  • 5. Tener un botón para reiniciar el juego o generar otro número.
  • 6. La interfaz debe verse ordenada y entendible.

Reglas:

  • • Pueden apoyarse en YouTube, internet o apuntes para investigar cómo hacerlo.
  • • La app debe ser funcional.
  • • Deben entregar evidencia de su trabajo.
  • • El trabajo es individual, salvo que se indique otra cosa.
